# Aquaticdle

A "wordl-" type game to guess a marine, aquatic, or semiaquatic mammal (currently excluding the semi-aquatic members of the sub-family sigmodontinae and the families muridae, thryonomyidae, and soricidae) based on 6 clues. Species refreshes at midnight
